# UI/UX Pro Max Skill

**Type:** Design System Generator  
**Repo:** [nextlevelbuilder/ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)  
**CLI:** `npm install -g uipro-cli`

## Overview

AI-powered reasoning engine that analyzes project requirements and generates a complete, tailored design system in seconds. Supports multiple platforms and frameworks.

## Key Features

- ✅ **Design System Generation** — pattern, style, colors, typography, effects
- ✅ **Multi-domain rules** — 10 categories (Tech, Finance, Healthcare, E-commerce, etc.)
- ✅ **67 UI Styles** — Glassmorphism, Claymorphism, Brutalism, Bento Grid, etc.
- ✅ **96 Color Palettes** — industry-specific
- ✅ **57 Font Pairings** — Google Fonts imports
- ✅ **Stack support** — React/Next.js, Vue/Nuxt, Svelte/Astro, SwiftUI, Flutter, Jetpack Compose
- ✅ **Pre-delivery checklist** — accessibility, responsiveness, anti-patterns

## Example Prompt

> Build a landing page for my beauty spa  
> → Generates: Soft UI Evolution, Calming palette, Cormorant Garamond/Montserrat, WCAG AA checklist

## Installation

```bash
npm install -g uipro-cli
uipro init --ai claude  # for Claude Code
```

## Usage

The skill activates automatically for UI/UX requests, or via slash command:

```
/ui-ux-pro-max Build a SaaS dashboard with dark theme
```
